OpenBangla Keyboard 1.0.0: Provat Released!

  • By Muhammad Mominul Huque
  • at September 8, 2016

We’re proudly announcing the release of OpenBangla Keyboard version 1.0.0 nicknamed Probhat(প্রভাত), an Open Source Bengali input method.

In Bangla Probhat(প্রভাত) means morning or starting of a new day. With this release we want to announce the starting of a new era of Bengali computing in Linux.

On the other hand, Probhat is the name of the most used Bengali keyboard layout in Linux and found in most distros. It is designed by Ekushey, the first project to distribute free Bengali typing tools for Windows users.

So OpenBangla Team wants to pay tribute to Ekushey team for being awesome!

This is the first stable version of OpenBangla Keyboard. So all things are new!
